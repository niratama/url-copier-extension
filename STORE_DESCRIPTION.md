# URL Copier - Smart URL & Title Copying

**Copy current page URL and Title in Markdown, HTML, or any custom format with one click!**

URL Copier is a powerful productivity tool that lets you copy the current tab's URL and Title in various formats. Perfect for developers, writers, and anyone who shares links frequently.

## Key Features
*   **Multiple Formats**: Comes with built-in templates for Markdown, HTML, and Plain Text.
*   **Custom Templates**: Create your own formats using `{{title}}` and `{{url}}` placeholders.
*   **Command Palette**: Press `Alt+P` to open a VSCode-style command palette to quickly search and select templates.
*   **Keyboard Shortcuts**:
    *   `Alt+C`: Copy using the last used template.
    *   `Alt+1` / `Alt+2`: Quick copy with your favorite templates.
*   **Context Menu**: Right-click to copy using any specific template.
*   **Copy All Tabs**: Copy links from all open tabs at once.
*   **Dark Mode Friendly**: The Command Palette automatically adapts to the page style.

## How to Use
1.  **Click the Icon**: Opens the Command Palette (or configure it to copy immediately).
2.  **Right-Click**: Select "URL Copier" to choose a specific format.
3.  **Shortcuts**: Use `Alt+C` for instant copy!

## Privacy
Your data (templates and settings) is stored locally in your browser. We do not collect or transmit any personal information.
