# URL Copier - スマートなURL＆タイトルコピー

**現在のページのURLとタイトルを、Markdown、HTML、または任意の形式でワンクリックコピー！**

URL Copierは、現在のタブのURLとタイトルを様々な形式でコピーできる強力な生産性向上ツールです。開発者、ライター、リンクを頻繁に共有する方に最適です。

## 主な機能
*   **複数のフォーマット**: Markdown、HTML、プレーンテキストのテンプレートを標準搭載。
*   **カスタムテンプレート**: `{{title}}` と `{{url}}` プレースホルダーを使って独自のフォーマットを作成可能。
*   **コマンドパレット**: `Alt+P` でVSCode風のコマンドパレットを開き、テンプレートを素早く検索・選択。
*   **キーボードショートカット**:
    *   `Alt+C`: 最後に使用したテンプレートでコピー。
    *   `Alt+1` / `Alt+2`: お気に入りのテンプレートでクイックコピー。
*   **コンテキストメニュー**: 右クリックから特定のテンプレートを選択してコピー。
*   **すべてのタブをコピー**: 開いているすべてのタブのリンクを一度にコピー。
*   **ダークモード対応**: コマンドパレットはページスタイルに合わせて自動的に適応。

## 使い方
1.  **アイコンをクリック**: コマンドパレットを開きます（または即座にコピーするように設定可能）。
2.  **右クリック**: 「URL Copier」を選択して特定のテンプレートを選びます。
3.  **ショートカット**: `Alt+C` で瞬時にコピー！

## プライバシー
データ（テンプレートや設定）はブラウザ内にローカルに保存されます。個人情報を収集・送信することはありません。
