# URL Copier Extension

現在のタブ（またはすべてのタブ）のURLとタイトルを、カスタマイズ可能な様々な形式でコピーできるChrome拡張機能です。

[English README is here](README.md)

## 機能

- **カスタマイズ可能なテンプレート**: `{{title}}` と `{{url}}` のプレースホルダーを使って独自のフォーマットを定義できます。
- **コンテキストメニュー統合**: ページ上で右クリックして、特定のフォーマットを選択できます。
- **コマンドパレットアクセス**: 拡張機能アイコンをクリックすると、コマンドパレットが開きます。
- **すべてのタブをコピー**: 現在のウィンドウで開いているすべてのタブのURLとタイトルをコピーするオプションがあります。
- **国際化対応**: 英語と日本語に対応しています。
- **キーボードショートカット**:
  - `Alt+C`: 最後に使用したテンプレートでコピーします。
  - `Alt+P`: コマンドパレットを開き、テンプレートを選択してコピーします。
  - `Alt+1`: スロット1に割り当てられたテンプレートでコピーします。
  - `Alt+2`: スロット2に割り当てられたテンプレートでコピーします。

## デフォルトテンプレート

- **Markdown**: `[{{title}}]({{url}})`
- **Markdown 2**: `{{title}} <{{url}}>`
- **HTML**: `<a href="{{url}}">{{title}}</a>`
- **Text**: `{{title}} {{url}}`
- **Two line text**:
  ```
  {{title}}
  {{url}}
  ```

## インストール方法

1.  このリポジトリをクローンまたはダウンロードします。
2.  Chromeを開き、`chrome://extensions/` にアクセスします。
3.  右上の **デベロッパーモード** を有効にします。
4.  **パッケージ化されていない拡張機能を読み込む** をクリックします。
5.  この拡張機能が含まれるディレクトリを選択します。

## 使い方

1.  ページ上で **右クリック** すると「URL Copier」メニューが表示されます。
2.  テンプレートを選択して、現在のページの情報をコピーします。
3.  **すべてのタブをコピー** を選択すると、ウィンドウ内の全タブの情報がコピーされます。
4.  ツールバーの拡張機能アイコンを **左クリック** すると、コマンドパレットが開きます。
5.  **オプション** ページ（アイコンを右クリック -> オプション）で、テンプレートの追加、編集、削除が可能です。
6.  **キーボードショートカット**: `Alt+P` でコマンドパレットを開き、すべてのテンプレートに素早くアクセスできます。オプションページで `Alt+1` と `Alt+2` にお気に入りのテンプレートを割り当てることも可能です。
